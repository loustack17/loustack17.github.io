# loustack17.github.io

Personal website built with Hugo static site generator.

## About

This is a static HTML website generated using [Hugo](https://gohugo.io/), a fast and flexible static site generator. The site is deployed to GitHub Pages.

## Development

To work on this site locally:

1. [Install Hugo](https://gohugo.io/installation/)
2. Clone this repository
3. Run `hugo server` to preview the site locally
4. Edit content in the `content/` directory
5. Customize layouts in the `layouts/` directory

## Building

To build the static HTML files:

```bash
hugo
```

This will generate static HTML files in the `docs/` directory, which is configured as the GitHub Pages source.

## Structure

- `content/` - Markdown content files
- `layouts/` - HTML templates
- `static/` - Static assets (images, CSS, JS)
- `docs/` - Generated static HTML (deployed to GitHub Pages)
- `hugo.toml` - Hugo configuration file
